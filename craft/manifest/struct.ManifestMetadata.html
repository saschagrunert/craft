General metadata about a package which is just blindly uploaded to the registry.

Note that many of these fields can contain invalid values such as the homepage, repository, documentation, or license. These fields are not validated by craft itself, but rather it is up to the registry when uploaded to validate these fields. Craft will itself accept any valid TOML specification for these values.
